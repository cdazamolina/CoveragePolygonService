# Coverage Polygon Service

Coverage Polygon Service es una servicio diseñado para la gestión de mapas de coberturas para rutas de transporte.

## Funcionalidades

* Crear, Leer, Editar y Eliminar poligonos de coberturas.
* Establecer la cobertura asignada a una localización geografica apartir de las cuberturas ya establecidas.
